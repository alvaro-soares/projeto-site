Curso de HTML5 com CSS3

A HTML é uma linguagem de marcação para a criação de sites. Atualmente na versão 5, ela usa a semântica dos elementos para criar o contéudo de uma página Web. Com as CSS, criamos as configurações visuais que um site vai ter.

Curso de javascript

JavaScript é uma linguagem de script criada para aumentar a interatividade entre um site e seus visitantes. Os códigos em JS são executados dentro do navegador e dão acesso aos elementos HTML por meio do DOM
